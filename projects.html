Phoneme Classification Deep Learning Model

Deep Learning, Python

Implemented a multilayer perceptron for frame-level speech recognition that maps raw MFCC features to phoneme class labels. The model learns both a feature representation and nonlinear decision boundary, trained end-to-end with cross-entropy loss to minimize the mismatch between output logits and ground-truth phonetic transcriptions via gradient descent.
